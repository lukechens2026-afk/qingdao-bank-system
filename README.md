
# 青岛银行日照分行 - 数字化触客原型系统

## 💻 本地预览指南 (重要)

由于浏览器对安全性（CORS）和 ES 模块的限制，**直接双击 index.html 无法运行**。请使用以下任一简单方法启动：

### 方法一：使用 VS Code (推荐)
1. 在 VS Code 中打开项目文件夹。
2. 安装扩展插件：**Live Server**。
3. 右键点击 `index.html`，选择 **"Open with Live Server"**。

### 方法二：使用命令行 (最快)
如果您安装了 Node.js，在项目根目录下运行：
```bash
npx serve
```
然后根据提示访问 `http://localhost:3000` 即可。

### 方法三：上传至 GitHub Pages
1. 将代码推送到 GitHub。
2. 在 Settings > Pages 中开启，即可获得一个**可分享给客户的在线链接**。
